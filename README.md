# Sorting-Hat-Quiz
A Python program that sorts users into Hogwarts houses based on their answers to a series of questions. Created to practice user input, variables, and conditional logic.

PURPOSE

Built as a fun beginner project to simulate an interactive quiz. This project demonstrates logic, scoring across multiple questions, and decision-making based on user input.

HOW TO RUN

- Install Python onto your system
- Download and run the file inside your terminal
- Follow the on-screen prompts and answer the questions
- The program will calculate your score, and sort you into a Hogwarts house!

CONCEPTS PRACTICED

- Variables and data storage
- Conditional logic with multiple branches
- User input and interaction
- Program flow and scoring system

FUTURE IMPROVEMENTS

- Add more questions and scoring variations
- Use lists or dictionaries to store questions and answers
